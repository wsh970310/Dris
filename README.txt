Variable parameters:
Max Data input: 18000 count per minute 
Gamma Image accumulate time: 30s
Gamma Image update period: 1s
Multi-thread mode: Intentservice

사용전:
1.DJI 개발자 계정으로 log in (아니면 켰을 때마다 꺼짐)
2.DJI development 홈페이지에서 mobile sdk(MSDK) key를 활성화 시켜야 됨
3.AndroidManifest.xml 에서 해당 key value 바꿈

사용중:
1. Wifi 연결 시켜야 함(DJI 계정 login 필요)
2. login success 떠야 버튼 누를 수 있음(아니면 login 정보 오류로 app꺼짐)
3. 나가서 실험할 때 : 먼저 wifi 연결 , 그다음 app 겨기. (지도 초기화 필요)
